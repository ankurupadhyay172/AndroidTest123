# 🛍️ Catch Android Assessment – Product Listing & Detail App

This project is my submission for the **Android Developer Assessment** at Catch Design. It demonstrates how to build a modern, scalable Android app using **Jetpack Compose**, **Jetpack Navigation**, **ViewModel**, and **Clean Code Architecture** principles.

<br/>

## 📱 Screens Overview

| Screen         | Description                                                                 |
|----------------|-----------------------------------------------------------------------------|
| 🔄 Loading     | Splash/loading screen with smooth transition                                |
| 🛒 PLP (Product Listing Page) | Displays product list fetched from remote JSON, styled as per Figma design |
| 📄 PDP (Product Detail Page)  | Shows selected product details beautifully and responsively     |

Design reference: [Figma Design Link](https://www.figma.com/file/Q8IUvJLAxPfR2FsomFhQSV/Catch-Design-–-iOS-Test?node-id=0%3A1)

<br/>

## 🚀 Features

✅ Built using **Jetpack Compose**  
✅ **ViewModel** for state management  
✅ **Navigation Component** for seamless screen transitions  
✅ **Remote JSON loading** using standard networking  
✅ Fully responsive and modern UI as per Figma  
✅ Modular code with **Clean Architecture** principles  
✅ Easy-to-read, scalable, and maintainable code  
✅ Git best practices followed  
✅ Ready to run in minutes
